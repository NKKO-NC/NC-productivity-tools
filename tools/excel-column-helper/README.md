# Excel Column Helper

The first shipped tool in this repo.

## What It Does

- Converts Excel column letters into numbers
- Expands a start and end range into a readable table
- Supports `A` through `XFD`
- Works in Traditional Chinese and English

## Files

- `index.html` for the tool page
- `app.js` for tool logic and page-level copy

## Why It Matters

This tool is a good baseline pattern for future repo additions because it has:

- simple shared styling
- bilingual copy
- pure logic that can later be extracted or tested
