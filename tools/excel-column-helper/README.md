# Excel 相對欄位對照

把 Excel 欄位區間轉成從 1 開始的相對序號。

Excel Relative Column Helper returns relative indexes starting from 1 for a selected column range.

## 範例

- `W ~ Z` → `1 ~ 4`
- `A ~ AC` → `1 ~ 29`

## 特色

- 支援 `A` 到 `XFD`
- 提供繁中 / English 介面切換
- 可獨立安裝成 PWA
- 支援基本離線快取
